# 💊 Maruthi Medical

A modern Flutter-based medical e-commerce application that allows users to browse medicines, search products instantly, manage their cart, and enjoy a clean, responsive shopping experience.

---

## ✨ Features

- 🔍 Instant medicine search
- 💊 Browse medicines by category
- 🛒 Add and remove medicines from the cart
- 💰 Automatic cart total calculation
- 🌙 Light & Dark mode support
- 📦 View medicine availability
- 🖼️ Product images
- 📱 Responsive and user-friendly interface
- ⚡ Smooth navigation and performance

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio or VS Code

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/maruthi-medical.git
```

2. Navigate to the project directory

```bash
cd maruthi-medical
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
